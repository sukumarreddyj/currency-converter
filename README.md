# 💱 Currency Converter

A simple and interactive Currency Converter web application built using **HTML**, **CSS**, and **JavaScript**. The application uses **real-time exchange rates** via the `ExchangeRate-API` to allow users to convert between different currencies instantly.

---

## 🔧 Features

- 🌐 Convert any currency to another with up-to-date exchange rates.
- ⏱ Real-time data fetched from ExchangeRate-API.
- 💡 Simple and clean UI for easy user experience.
- 📱 Responsive design for mobile and desktop devices.

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **API**: [ExchangeRate-API](https://www.exchangerate-api.com/)

---

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/sukumarreddyj/currency-converter.git
cd currency-converter
````

### 2. Open `index.html` in your browser

You can directly open the file in your browser or use a local server.
